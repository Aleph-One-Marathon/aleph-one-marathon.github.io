Download

  • Download for Mac OS X
  • The disk image should automatically mount
  • Copy the {{ include.game }} app to your Applications folder
  • Launch {{ include.game }}
  • Download for Windows
  • Open the zip archive
  • Copy the {{ include.game }} folder to your Program Files folder
  • Launch {{ include.game }}.exe
  • Download Aleph One for Linux
  • Unpack the tar.bz2 file
  • Run ./configure && make && make install
  • Download {{ include.game }} game files
  • Place the unzipped files in /usr/local/share/AlephOne
  • Run alephone

{{ include.game }} is just one of the many games you can play with Aleph One. If you already have a copy of the Aleph One application, you can download the standalone {{ include.game }} game files for use with Aleph One 1.0 or later.
